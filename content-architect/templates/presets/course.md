# Course Structure Preset

## Terminology
- **Sections** = Modules
- **Units** = Lessons
- **Opening** = Welcome / Course Introduction
- **Closing** = Course Wrap-up / Final Assessment

## Typical Structure

### Course Introduction
1. Welcome message
2. Course overview and objectives
3. Prerequisites and expectations
4. How to get the most from this course

### Modules
- 4-12 modules, each covering a major topic
- Each module contains 3-7 lessons
- Each lesson: 15-45 minutes of content
- Total: 10-60 hours depending on depth

### Module Structure
1. Module introduction (objectives, overview)
2. Lessons (core content)
3. Module summary and review
4. Assessment (quiz, assignment, project)

### Course Wrap-up
1. Course summary and review
2. Final project or assessment
3. Next steps and further resources
4. Certificate/completion criteria

## Architecture Patterns

### Progressive Skill Building
Best for: technical skills, languages, instruments
- Each module builds on the previous
- Increasing difficulty and complexity
- Prerequisites enforced between modules

### Topic-Based
Best for: broad subjects, survey courses
- Each module covers a distinct topic
- Can be consumed somewhat independently
- Connected by overarching theme

### Project-Based
Best for: creative skills, applied learning
- Each module contributes to a larger project
- Theory interleaved with hands-on practice
- Final deliverable at course end

### Cohort/Live
Best for: coaching, mastermind, interactive programs
- Weekly modules released on schedule
- Live sessions (Q&A, workshops, coaching)
- Peer interaction and accountability

## Lesson Template
1. **Learning objectives** (what they'll be able to do)
2. **Core content** (teaching, demonstration, examples)
3. **Practice/Exercise** (hands-on application)
4. **Key takeaways** (summary of main points)
5. **Action item** (what to do before next lesson)

## Duration Guidelines
- Lesson: 15-45 minutes content
- Module: 2-8 hours total
- Full course: 10-60 hours
- Mini-course: 2-5 hours
